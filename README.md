# Movies App — JavaScript

A movie discovery app built with vanilla JavaScript that fetches data from the TMDB (The Movie Database) API and displays trending and top-rated films.

## Features

- Browse trending and top-rated movies
- Movie detail view: poster, rating, overview, release date
- Search movies by title
- Responsive card-based layout

## Running

1. Get a free API key from [TMDB](https://www.themoviedb.org/settings/api)
2. Add your API key to the config in the JS source
3. Open `index.html` in any browser

## Tech Stack

Vanilla JavaScript, Fetch API, TMDB API, HTML5, CSS3
